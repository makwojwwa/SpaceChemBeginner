Production assignment interface


Production assignment interface with annotations

  1. Input structure

    It produces input atoms/molecules which you must use to obtain the required molecule.

  2. Input note

    This is a small representation of the input area grid, which shows what atom/molecule will be input and in which grid cell.

  3. Output structure

    It accepts the required molecules.

  4. Required molecule

    This is your target molecule in a production assignment. Accompanied by a molecule counter, which shows the progress of the assignment.

  5. Available structures

    This section includes structures which you can place in a production assignment.

  6. Reactor toggle

    You can choose between empty reactors and any pre-programmed reactors you saved during previous assignments.

  7. Obstacle

    No structures or pipelines can be placed on obstacles. You can recognize them by the absence of grid cells.

  8. Reactor quota

    Shows the number of structures you are allowed to place in a production assignment. Filled bars show the number of structures you are using now.

  9. Info box

    Mouse over a structure to see an info box containing additional information and tips.

  10. Stub pipes

    The one on the left side is for the input, while the one on the right side is for the output.

  11. Pipeline

    Used for connecting input structures with reactors and output structures.

  12. Output note

    Helps you remember what intermediate molecules are output at which reactor. You should create it yourself.

  13. Progress counter

    Shows the number of cycles, symbols and reactors used in your solution, as well as the overall progress of your solution.
