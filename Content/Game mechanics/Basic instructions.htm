Basic instructions

Instruction Icon Description
Start The first and the most important instruction of all. It marks the place where the waldo begins executing the algorithm.
Input This instruction makes an atom/molecule appear in the input area, in the grid cell that is indicated in the input note presented to the left of the reactor grid.
Output This instruction makes an atom/molecule disappear in the output area.
Grab/Drop There are three variants of the GRAB DROP button. The default one makes the waldo grab or drop an atom/molecule located exactly at the place where the instruction is located. The button can be switched to either GRAB or DROP so that the waldo can perform only one operation (grabbing or dropping) instead of both of them.
Rotate molecule This instruction is responsible for rotating the molecule grabbed by the waldo 90 degrees clockwise or counter-clockwise. It does not rotate single atoms.
Bond+/Bond- These instructions are variants of the BOND+ instruction button. BOND+ creates a bond between atoms that are present on adjacent bonders, whereas BOND- destroys a bond between atoms that are present on adjacent bonders. The instruction does not have to be placed directly on bonders.
Arrow instructions The arrows force the waldos to change the direction of their paths in the reactor.
Sync SYNC is useful for avoiding collisions and making sure that both waldos execute their respective algorithms in the proper sequence. You have to place one SYNC for the red waldo and one for the blue waldo to achieve this effect. The first waldo reaching its SYNC will stop there and wait until the other waldo reaches its SYNC. Once this happens, the first waldo starts running again.
